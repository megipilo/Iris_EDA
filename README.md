# Iris Dataset - Exploratory Data Analysis

## About the assignment

In this assigment, I performed exploratory data analysis (EDA) on the Iris dataset. The goal was to understand the structure of the dataset, explore relationships between features, and identify patterns between different flower species.

## Dataset

The dataset contains 150 samples of iris flowers with the following features:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width
* Species (target variable)

## Analysis Steps

* Loaded and explored the dataset
* Checked dataset shape and basic information
* Analyzed distributions of features
* Visualized relationships between variables using plots
* Compared feature values across different species

## Visualizations

* Histograms to understand feature distributions
* Scatter plots to observe relationships between variables
* Pairplots to compare all features together
* Boxplots to analyze variation between species

## Goal

The goal of this analysis was to better understand how the features differ between iris species and to prepare the dataset for potential machine learning tasks.

## Tools

* Python
* Pandas
* Matplotlib
* Seaborn
